# Offer

## Positioning

Do not pitch this as an app generator.

Pitch it as:

> We help small businesses replace messy manual workflows with simple internal tools and clearer tracking.

## What You Are Selling

- workflow rescue
- operational clarity
- less admin work
- fewer dropped tasks
- faster response times

## Example Before / After

### Service business

Before:

- email
- spreadsheet
- phone calls

After:

- request form
- dashboard
- status tracking
- automated notifications

### Property manager

Before:

- maintenance requests arrive through scattered channels
- updates get lost in email

After:

- request intake form
- maintenance queue
- status dashboard
- tenant communication flow

### Small manufacturer

Before:

- Excel-based inventory tracking
- manual reorder awareness

After:

- inventory dashboard
- low-stock alerts
- reporting

## Suggested Early Offer Structure

1. Workflow discovery package
2. Implementation plan
3. Tool build
4. Optional support retainer

## Suggested Discovery Deliverable

Give the client:

- current-state summary
- problems and bottlenecks
- recommended workflow
- recommended tool scope
- estimated timeline
- rough budget range
