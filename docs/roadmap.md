# Roadmap

## Phase 0

Manual consulting with AI-assisted workflow discovery.

Flow:

```text
Lead -> Form -> Summary -> Spec -> Review -> Quote -> Build
```

Primary goal:

- get the first paying customer

## Phase 1

Spec generator web app.

Input:

- business process description

Output:

- business summary
- problems
- entities
- workflow
- dashboards
- suggested solution
- estimated build effort

Primary goal:

- reduce your turnaround time and standardize proposals

## Phase 2

Internal builder.

Generate:

- schema
- forms
- CRUD screens
- workflows
- reports

Primary goal:

- compress build time for repeated use cases

## Phase 3

Deployable apps.

Generate:

- customer portal
- admin dashboard
- notifications
- reports

Primary goal:

- move from services into productized delivery

## Phase 4

Template marketplace.

Examples:

- service requests
- work orders
- inventory tracking
- appointment scheduling
- customer intake
- job tracking

Primary goal:

- product leverage and repeatability
